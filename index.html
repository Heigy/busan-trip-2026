📍 嵌入 Google My Maps

網站使用 Google My Maps（免費，無需 API Key）。請先建立地圖並把連結填入 config.js。

  1. 打開 Google My Maps
  2. 建立新地圖 → 匯入 → 上傳 busan-locations.kml / jeju-locations.kml
  3. 分享 → 「知道連結的任何人可查看」
  4. 點 ⋮ → 「在地圖上嵌入」→ 複製 src="..." 裡的網址
  5. 貼到 config.js 的 embedUrl，分享連結貼到 viewUrl
  6. git push 重新部署

完成後刷新此頁，左側會顯示 My Maps，右側為行程列表。
